\doxysubsection{DESCRIPTION}\label{rnaprediction_8cpp_DESCRIPTION}
The following program takes a character array representing the RNA sequence as input and prints the number of bases pairs that are present in the secondary structure of the sequence. Rules for forming bases pairs are as follows\+:
\begin{DoxyEnumerate}
\item Pairs of bases match up; Each base matches with 1 other base.
\item Adenine always matches with Uracil.
\item Cytosine always matches with Guanine.
\item There are no kinks in the folded molecule.
\item Structures are knot-\/free.
\item No sharp turns allowed 
\end{DoxyEnumerate}
