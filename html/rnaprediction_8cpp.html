DESCRIPTION

The following program takes a character array representing the RNA sequence as input and prints the number of bases pairs that are present in the secondary structure of the sequence. Rules for forming bases pairs are as follows:

  1. Pairs of bases match up; Each base matches with 1 other base.
  2. Adenine always matches with Uracil.
  3. Cytosine always matches with Guanine.
  4. There are no kinks in the folded molecule.
  5. Structures are knot-free.
  6. No sharp turns allowed
